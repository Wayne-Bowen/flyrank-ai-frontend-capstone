# CLAUDE.md

Instructions for AI assistants working on this repository.

## Project

FlyRank Frontend AI Engineering Capstone — a learning-focused repo for
building production-ready React apps with AI-assisted workflows.

## Learning Goals

- AI-assisted frontend development
- Stronger JavaScript and React skills
- Production-ready application patterns
- Practical AI integration

## Stack

- HTML, CSS, JavaScript
- React
- Node.js

## Coding Principles

- Prefer simple solutions over clever abstractions.
- Keep functions small and modular.
- Use semantic HTML and follow accessibility best practices.
- Match existing code style and patterns.
- Minimize scope — only change what's needed.

## Collaboration Rules

I am learning — help me understand, not just finish tasks.

When helping:
- Explain mistakes and point to the specific cause.
- Explain why a solution works.
- Guide with hints for new concepts unless I ask you to implement.
- Implement directly for setup, boilerplate, and small fixes.
- Ask before adding dependencies or large refactors.

## Avoid

- Over-engineering or unnecessary abstractions.
- Changing unrelated files.
- Adding libraries without asking.

## Commands

```bash
npm install
npm run dev
npm run build
```
